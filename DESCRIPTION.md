# Shalaye - Content Simplifier

A Chrome extension that simplifies web content using Google Gemini AI. One-click summarization with adjustable complexity levels, key takeaways extraction, text-to-speech, and saved summaries.

## Key Features
- 🚀 Powered by Google Gemini 2.0 Flash
- 📝 One-click page summarization
- 🎚️ Adjustable complexity (Quick/Standard/Detailed)
- ✓ Key takeaways extraction
- 🔊 Text-to-speech
- 💾 Save & organize summaries
- 🌙 Light/Dark mode toggle
- 💡 Select text → instant explanations

## Tech Stack
- Chrome Extension (Manifest V3)
- Google Gemini API (primary)
- Groq API (failover)
- Vanilla JS/CSS

Built for the Gemini API Hackathon 🏆
